Delete submodule

Via console

  1. Delete a submodule folder.

    git rm
    path

  2. Commit already staged changes.

    git commit -m "Deleted submodule."

Manually

  1. From a /.gitmodules file delete section of appropriate submodule. Or delete the file if no more submodules.

  2. From a .git/config file delete section of appropriate submodule.

  3. Delete the folder of submodule.

  4. Stage and commit.

  5. (Optionally.) From a /.git/modules folder delete a folded of appropriate submodule.
